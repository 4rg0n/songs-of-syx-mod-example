# Animal

`data/assets/init/animal`

| Key                    | Required | Default     | Min   | Max     | Description                                                                                                                                                                                 | Example                                                                                                                        |
|------------------------|----------|-------------|-------|---------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| COLOR                  | no       | 127_127_127 | none  | none    | The base color. The game will vary the color little bit per spawned animal.                                                                                                                 | `255_255_255` or<br/> `{R: 255, G: 255, B: 255, }`                                                                             |
| ICON                   | no       | dummy icon  | none  | none    | Path to the sprite tile used for the icon. The path will look into `assets/init/sprite/icon`.                                                                                               | `32->animal->Auroch->0`<br/>The real path would be:<br/>`assets/init/icon/32/animal/Auroch.png` first (0) tile                 |
| SPRITE                 | yes      | none        | none  | none    | Name of the sprite file in `data/assets/init/sprite/animal` (without file ending).                                                                                                          | `Auroch`                                                                                                                       |
| ANIMAL_CALL_\<ANIMAL\> | yes      | none        | none  | none    | List of sounds the animal will make. `<ANIMAL>` is the file name of the animal e.g. `ANIMAL_CALL_AUROCH` for `AUROCH.txt`.<br/>Available sounds can be found in `assets/audio/mono/animal`. | `[ animal->Bull*, ]`<br/>The `*` is a wildcard.<br/>It will use all files in `/assets/audio/mono/animal` beginning with `Bull` |
| SPEED                  | yes      | none        | 1     | 31      | Movement / acceleration speed in tiles.                                                                                                                                                     |                                                                                                                                |
| MASS                   | yes      | none        | 1     | 500     | Minimum weight in "stones". It will be used when calculating the damage the animal does.                                                                                                    |                                                                                                                                |
| HEIGHT                 | yes      | none        | 0     | 50      | Height from ground level.                                                                                                                                                                   |                                                                                                                                |
| DANGER                 | yes      | none        | 0.0   | 1.0     | Chance for the animal to attack a citizen.                                                                                                                                                  |                                                                                                                                |
| DAMAGE                 | yes      | none        | 0.0   | 10000.0 | Which damage type and how much damage the animal does with a collision.<br/>See file names in `assets/init/stats/damage` for possible damage types.                                         | `{ PIERCE: 0.4, }`</br>or `{ *: 0.4, }` for all                                                                                |
| CARAVAN                | no       | false       | none  | none    | Whether the animal can be used in caravan.                                                                                                                                                  |                                                                                                                                |
| RESOURCES              | yes      | none        | none  | none    | List with resources the animal drops when killed.<br/>See file names in `assets/init/resource` and sub folders for possible values.                                                         | `[MEAT, LEATHER,],`                                                                                                            |
| RESOURCE_AMOUNT        | yes      | none        | none  | none    | List with amounts for each dropped resource.                                                                                                                                                | `[0.8, 0.5,],`                                                                                                                 |
| LIVES_IN_CAVES         | yes      | none        | 0.0   | 1.0     | Chance for the animal to spawn in mountains.                                                                                                                                                |                                                                                                                                |
| CLIMATE.\<CLIMATE\>    | yes      | none        | 0.0   | 1.0     | Chance for the animal to spawn in specific climates.<br/>Possible \<CLIMATE\> names can be found in `assets/init/config/CLIMATE.txt`.                                                       | `{ COLD: 0.8, TEMPERATE: 1.0, HOT: 0.0, }`</br>or `{ *: 1.0, }` for all                                                        |
| TERRAIN.\<TERRAIN\>    | yes      | none        | 0.0   | 1.0     | Chance for the animal to spawn in specific terrains.<br/>Possible \<TERRAIN\> names can be found in `assets/init/config/TERRAIN.txt`.                                                       | `{ FOREST: 0.5, WET: 0.5, NONE: 1.0, }`</br>or `{ *: 1.0, }` for all                                                           |
