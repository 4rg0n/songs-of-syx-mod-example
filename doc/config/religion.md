# Religion

`data/assets/init/religion`

| Key                     | Required | Default     | Min  | Max                                                                                           | Description                                                                                                         | Example                                                                                                              |
|-------------------------|----------|-------------|------|-----------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| NAME                    | yes      | none        | none | none                                                                                          | Name of the religion.                                                                                               | `Aminionism`                                                                                                         |
| DEITY                   | yes      | none        | none | none                                                                                          | Name of the deity of the religion.                                                                                  | `Aminion`                                                                                                            |
| DESC                    | yes      | none        | none | none                                                                                          | Short description about the religion.                                                                               | `Aminion, the lord of chaos promotes submission and his temples accepts human sacrifices (prisoners).`               |
| DEFAULT_SPREAD          | yes      | none        | 0.0  | [Double.MAX_VALUE](https://docs.oracle.com/javase/8/docs/api/java/lang/Double.html#MAX_VALUE) | How likely it is for people to convert to this religion                                                             |                                                                                                                      |
| COLOR                   | no       | 127_127_127 | none | none                                                                                          | Used for coloring the religion in the UI.                                                                           | `255_255_255` or<br/> `{R: 255, G: 255, B: 255, }`                                                                   |
| ICON                    | no       | dummy icon  | none | none                                                                                          | File path and sprite tile to the icon found in `assets/sprite/icon/`.                                               | `32->religion->AMINION->0`<br/>The real path would be:<br/>`assets/init/icon/32/religion/AMINION.png` first (0) tile |

| BOOST.\<KEY\>           | no       | none        | none | none                                                                                          | See [boosts](boost)                                                                                                 |                                                                                                                      |

Texts for religion can be found in `assets/text/religion`.
