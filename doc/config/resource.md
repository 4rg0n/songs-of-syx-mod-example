# Resources

`data/assets/init/resource`

| Key              | Required | Default     | Min  | Max   | Description                                                                                                              | Example                                                                                                              |
|------------------|----------|-------------|------|-------|--------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| ICON             | no       | dummy icon  | none | none  | Path to the sprite tile used for the icon. The path will look into `assets/init/sprite/icon`.                            | `24->resource->Alcohol->0`<br/>The real path would be:<br/>`assets/init/icon/32/resource/Alcohol.png` first (0) tile |
| COLOR            | no       | 127_127_127 | none | none  | Used for tint the resource when layered in a pile.                                                                       | `255_255_255` or<br/> `{R: 255, G: 255, B: 255, }`                                                                   |
| MINIMAP_COLOR    | no       | 127_127_127 | none | none  | For displaying the resource on the minimap and when max zoomed out.                                                      | `255_255_255` or<br/> `{R: 255, G: 255, B: 255, }`                                                                   |
| DEGRADE_RATE     | yes      | none        | 0.0  | 1.0   | How fast a resource will degrade.                                                                                        |                                                                                                                      |
| SPRITE           | yes      | none        | none | none  | Name of the sprite file in `data/assets/init/sprite/resource` (without file ending).                                     | `Livestock`                                                                                                          |
| SPRITE_DEBRIS    | yes      | none        | none | none  | Name of the sprite file used when the resource is destroyed in `data/assets/init/sprite/resource` (without file ending). | `Wood`                                                                                                               |
| CATEGORY_DEFAULT | yes      | none        | 0    | 3     | Determines the display order in UIs.                                                                                     |                                                                                                                      |
| PRICE_CAP        | no       | 1.0         | 0.0  | 1.0   | Multiplier for buying the resource.                                                                                      |                                                                                                                      |
| PRICE_MUL        | no       | 1.0         | 0.0  | 100.0 | Multiplier for selling the resource.                                                                                     |                                                                                                                      |
